## Version 2.0

Five datasets were released in v2.0. They span two neuron models (STG and DA), two conductance-sampling strategies (DICs and Monte Carlo), and two simulation modes (noise-free and noisy current injection).

---

### 1. `dics_stg_nonoise`

| Property | Value |
|----------|-------|
| **Neuron model** | STG (Liu et al., 1998) |
| **Conductance sampling** | DICs framework |
| **Noise injection** | None (deterministic ODE) |
| **Ion channels** | Na, Kd, CaT, CaS, KCa, A, H, leak |
| **DICs parameters** | g_s in [-20, 20], g_u in [0, 20], V_th = -51.0 mV |
| **Simulation window** | 6 000 ms total - 3 000 ms transient discarded → 3 000 ms recorded |
| **Files** | `training_set_stg.csv` (1 199 999 rows), `validation_set_stg.csv` (192 000 rows), `test_set_stg.csv` (192 000 rows) |

Conductances are sampled using the Dynamic Input Conductances (DICs) framework, which generates **degenerate populations**: neurons that produce the same activity pattern despite having widely different individual conductance values. Each group of neurons shares the same (g_s, g_u) neuromodulation point, ensuring they are functionally equivalent by construction. The simulation is fully deterministic (no noise). This is the most complete split, providing train / validation / test sets.

### 2. `dics_stg_noise_5`

| Property | Value |
|----------|-------|
| **Neuron model** | STG (Liu et al., 1998) |
| **Conductance sampling** | DICs framework |
| **Noise injection** | Band-limited Gaussian, sigma = 5.0 uA/cm2, cutoff 1 000 Hz |
| **Ion channels** | Na, Kd, CaT, CaS, KCa, A, H, leak |
| **DICs parameters** | g_s in [-20, 20], g_u in [0, 20], V_th = -51.0 mV |
| **Simulation window** | 6 000 ms total - 3 000 ms transient discarded → 3 000 ms recorded |
| **Files** | `stg_dataset_train.csv` (998 499 rows), `stg_dataset_val.csv` (199 350 rows) |

Same conductance sampling as `dics_stg_nonoise` - degenerate populations generated by the DICs framework - but a band-limited Gaussian current (sigma = 5 uA/cm2, low-pass filtered at 1 000 Hz) is injected during simulation. This makes the spike trains stochastic and tests robustness to in-vivo-like noise. Train and validation splits are provided.

---

### 3. `dics_da_nonoise`

| Property | Value |
|----------|-------|
| **Neuron model** | DA (Qian et al., 2014) |
| **Conductance sampling** | DICs framework |
| **Noise injection** | None (deterministic ODE) |
| **Ion channels** | Na, Kd, CaL, CaN, ERG, NMDA, leak |
| **DICs parameters** | g_s in [-10, 15], g_u in [0, 20], V_th = -55.5 mV |
| **Simulation window** | 12 000 ms total - 3 000 ms transient discarded → 9 000 ms recorded |
| **Files** | `training_set_da.csv` (1 215 276 rows), `validation_set_da.csv` (467 000 rows) |

Midbrain dopamine neuron model with conductances sampled via the DICs framework, generating **degenerate populations** of DA neurons that share the same functional activity despite differing conductance sets. The longer simulation window (9 000 ms of effective recording) captures the slower dynamics of the DA model. The simulation is deterministic. Train and validation splits are provided.

### 4. `dics_da_noise_5`

| Property | Value |
|----------|-------|
| **Neuron model** | DA (Qian et al., 2014) |
| **Conductance sampling** | DICs framework |
| **Noise injection** | Band-limited Gaussian, sigma = 5.0 uA/cm2, cutoff 1 000 Hz |
| **Ion channels** | Na, Kd, CaL, CaN, ERG, NMDA, leak |
| **DICs parameters** | g_s in [-10, 15], g_u in [0, 20], V_th = -55.5 mV |
| **Simulation window** | 12 000 ms total - 3 000 ms transient discarded → 9 000 ms recorded |
| **Files** | `da_dataset_train.csv` (198 659 rows), `da_dataset_val.csv` (81 121 rows) |

Same as `dics_da_nonoise` - degenerate DA neuron populations from the DICs framework - but with band-limited Gaussian noise injected during simulation (sigma = 5 uA/cm2, 1 000 Hz cutoff). Train and validation splits are provided.

### 5. `mc_gamma_stg_noise_5`

| Property | Value |
|----------|-------|
| **Neuron model** | STG (Liu et al., 1998) |
| **Conductance sampling** | Monte Carlo - Gamma distribution |
| **Noise injection** | Band-limited Gaussian, sigma = 5.0 uA/cm2, cutoff 1 000 Hz |
| **Ion channels** | Na, Kd, CaT, CaS, KCa, A, H, leak |
| **Conductance ranges (mS/cm2)** | g_Na in [0, 8000], g_Kd in [0, 350], g_CaT in [0, 12], g_CaS in [0, 50], g_KCa in [0, 250], g_A in [0, 600], g_H in [0, 0.7], g_leak in [0, 0.02] |
| **Simulation window** | 6 000 ms total - 3 000 ms transient discarded → 3 000 ms recorded |
| **Files** | `stg_dataset_train.csv` (845 325 rows), `stg_dataset_val.csv` (149 175 rows) |

Conductances are sampled independently using a Gamma distribution whose mean and variance match those of the corresponding uniform range. Unlike the DICs-based datasets, there is no degeneracy structure - channels are sampled independently with no constraint linking their values - this is a broad, unconstrained exploration of the conductance space. Noise is injected during simulation. Train and validation splits are provided.
